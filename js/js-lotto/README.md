### 기능 요구 사항

- [x] 사용자에게 구매금액을 입력받고 구매 금액에 따른 개수만큼 로또를 발행한다.
- [x] 정답 로또(WinningLotto)를 발행한다.
- [x] 보너스 번호를 입력받는다.
- [x] 정답 로또, 보너스 번호와 사용자가 발행한 로또를 비교하여 당첨 통계를 만든다.
- [x] 총 당첨금과 구매 금액을 통해 수익률을 계산한다.
- [x] 사용자에게 통계와 수익률을 출력한다.

### Think

- Lotto 클래스를 이용하여 정답 로또와 사용자가 발행한 로또를 만든다.
- 정답 로또에는 보너스 숫자도 있기 때문에 WinningLotto 클래스를 만들어 관리한다.
